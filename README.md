# Lab 6 - Data Analysis with Pandas

## Overview
This lab focuses on practicing data analysis techniques using the pandas library in Python.

## Purpose
The goal of this project is to develop and enhance skills in data manipulation, cleaning, and analysis using pandas.

## Key Learning Objectives
- Loading and exploring datasets with pandas
- Data cleaning and preprocessing
- Data filtering and selection
- Grouping and aggregating data
- Creating summary statistics
- Data visualization with pandas

## Technologies Used
- Python
- pandas
- Jupyter Notebook / Python scripts

## Skills Practiced
- Data manipulation with pandas DataFrames
- Handling missing data
- Data transformation and reshaping
- Exploratory Data Analysis (EDA)

## How to Use
1. Clone this repository
2. Install required dependencies: `pip install pandas`
3. Run the Jupyter notebooks or Python scripts
